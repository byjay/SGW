# 그룹웨어 앱 알림 및 배포 기능 개선

## 🎯 개요
앱 로그인 상태에서 브라우저 푸시 알림과 배포 상태 알림을 추가하여 사용자 경험 개선

## ✨ 추가된 기능

### 1. 브라우저 알림 시스템
- **NotificationContext**: 전역 알림 상태 관리
- **NotificationCenter**: 알림 센터 UI
- 알림 유형:
  - 📢 공지사항 (notice)
  - 💬 메시지 (message)
  - ✅ 결재 (approval)
  - 📅 연차 신청 (leave)
  - 🚀 배포 상태 (deploy)

### 2. 알림 권한 관리
- 브라우저 알림 권한 요청
- 알림 켜기/끄기 설정
- 권한 거부 시 안내 메시지

### 3. 알림 센터 기능
- 실시간 알림 목록 표시
- 읽음/안읽음 상태 관리
- 알림 삭제 (개별/전체)
- 알림 클릭 시 해당 화면으로 이동
- 시간 포맷팅 (방금, N분 전, N시간 전, 날짜)

### 4. 배포 알림 추가
- 배포 완료 시 알림
- 배포 URL 제공
- 배포 상태 표시 (성공/실패)

## 📱 사용 방법

### 알림 센터 접근
- 상단 알림 아이콘 클릭
- 읽지 않은 알림 수량 표시 (배지)
- 알림 목록 드롭다운

### 알림 설정
- 설정 아이콘 클릭
- "알림 사용" 토글
- 브라우저 권한 요청 팝업 표시

### 알림 확인
- 알림 항목 클릭 시 읽음 처리
- 해당 화면으로 자동 이동
- 삭제 아이콘으로 개별 삭제

## 🔧 기술 구현

### 파일 구조
```
그룹웨어/
├── contexts/
│   └── NotificationContext.tsx      # 전역 알림 상태
├── components/
│   └── NotificationCenter.tsx    # 알림 센터 UI
└── App.tsx                           # NotificationProvider 추가
```

### API 사용
- **Notification API**: 브라우저 푸시 알림
- **localStorage**: 알림 저장 및 설정
- **React Context**: 전역 상태 관리

### 알림 권한
- `granted`: 알림 허용됨
- `denied`: 알림 거부됨
- `default`: 권한 선택 안 함

## 🎨 UI/UX

### 디자인 원칙
- 직관적인 아이콘 (타입별 색상)
- 실시간 배지 업데이트
- 부드러운 애니메이션
- 반응형 레이아웃

### 색상 코드
- **공지사항**: 빨간색 (red)
- **메시지**: 에메랄드색 (emerald)
- **결재**: 파란색 (blue)
- **연차**: 보라색 (purple)
- **배포**: 주황색 (orange)

## 🚀 배포 알림 활용

### 배포 상태 알림
```typescript
// 배포 성공 알림
addNotification({
  type: 'deploy',
  title: '배포 완료',
  content: '최신 버전이 배포되었습니다.',
  actionUrl: '#home'
});
```

### 배포 실패 알림
```typescript
addNotification({
  type: 'deploy',
  title: '배포 실패',
  content: '배포 중 오류가 발생했습니다.',
  actionUrl: '#home'
});
```
